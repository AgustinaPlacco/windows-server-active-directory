# Demostración en vídeo

Como parte de este proyecto se realizó una demostración práctica en vídeo donde se muestra el funcionamiento de la infraestructura implementada.

Durante la demostración se verifican las principales configuraciones realizadas en el controlador de dominio Windows Server 2022 y en el equipo cliente Windows 11, comprobando el correcto funcionamiento de Active Directory y de los servicios configurados.
